# 弹幕保护计划

2020年05月21日，B站启用protobuf弹幕，许多高级弹幕，代码弹幕等丢失并无法通过正常手段找回来。本目录下是部分曾经弹幕艺术的备份，希望能通过这种手段，让这些上古珍宝不至于埋没。
